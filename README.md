# vues-plugin-qgis

Vue is a python plugin for view management integrated in the QGIS status bar.

## Pré-requis

- qgisMinimumVersion=3.4

## Fonctionnalités

French :
* Visualiser des styles prédéfinis​
* Créer un style prédéfini​
* Importer / exporter un style​
* Modifier un style prédéfini

English : 
* view predefined styles
* create predefined style
* Import / export a style
* Edit a predefined style
